QR-TOTP Authenticator

Python Google Authenticator-style QR generator and 6-digit TOTP checker

Python Complete

Overview

Built in Python 3 to mimic core Google Authenticator workflows. The project generates QR codes for scanner-based enrollment and produces the same six-digit TOTP flow used by the authenticator algorithm.

Highlights

  • QR code generation for scanner onboarding
  • 6-digit TOTP generation flow
  • Python-based security-focused coursework project

Tech Stack / Focus

Python 3, PyQRCode, TOTP concepts, QR provisioning
